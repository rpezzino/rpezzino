# Conjunto de Dados: Clima_INMET

## 1. Descrição
Conjunto de dados meteorológicos históricos provenientes do INMET.

## 2. Granularidade
- Temporal: diária
- Espacial: município

## 3. Campos Principais
- data
- precipitacao_mm
- temperatura_media

## 4. Atualização
Atualização automática periódica.

## 5. Limitações
- Falhas de cobertura em alguns municípios
- Dados ausentes em determinados períodos
